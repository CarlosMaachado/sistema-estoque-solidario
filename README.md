Dashboard de Gestão de Estoque Solidário


Interface interativa desenvolvida para otimizar o controle de donativos da Igreja da Salvação , unindo design intuitivo e análise de dados para facilitar o trabalho voluntário e a transparência logística.

 Tecnologias Utilizadas
HTML5 : Estruturação semântica de dados.

CSS3 (Flexbox) : Layout responsivo adaptado para dispositivos móveis.

JavaScript (ES6+) : Lógica de negócios e manipulação dinâmica do DOM.

Chart.js : Visualização de dados em tempo real para monitoramento de criticidade.

 Funcionalidades Principais
Monitoramento por Criticidade : Dashboards dinâmicos que alteram a cor com base no nível do estoque (Crítico, Moderado e Abastecido).

Gestão de Movimentação : Sistema de registro de entradas e saídas com validação de dados.

Histórico Auditável : Tabela de logs com registro automático de dados e hora (Timestamp) para cada ação.

Filtro Inteligente : Busca dinâmica no histórico de movimentações para facilitar a consulta rápida.

 Objetivo Acadêmico
Este projeto foi desenvolvido como parte de minha trajetória na Universidade São Judas , passando a aplicar conceitos de lógica de programação em soluções de impacto social real. Como estudante de Publicidade e Propaganda , foquei na criação de uma ferramenta que não apenas processa dados, mas comunica a urgência e o status das necessidades comunitárias de forma clara e visual.
